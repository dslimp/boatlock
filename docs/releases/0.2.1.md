# BoatLock Firmware 0.2.1 Release Notes

Release date: 2026-04-27

## Highlights

- Added the standard `nh02` deploy command for the normal moved-hardware path:
  `tools/hw/nh02/deploy.sh`.
- Fixed the app-to-firmware OTA bridge after BLE live-frame schema changes: the
  release app can decode the previous v3 live frame long enough to start the
  update to the current v4 firmware.
- Split steering setup into motor-side steps per revolution and gear ratio. The
  current DRV8825 full-step hardware defaults to `StepSpr=200` and
  `StepGear=36`.
- Raised default stepper speed and acceleration so manual steering no longer
  crawls after applying the gearbox ratio.
- Kept manual control on the atomic `MANUAL_TARGET` deadman path and removed
  legacy split manual commands from the accepted surface.

## Validation Snapshot

- Native firmware tests: pass.
- Flutter analyze and tests: pass.
- CI helper tests: pass.
- `nh02` phone BLE OTA deploy: pass with post-update telemetry recovery.

## Upgrade Notes

- Firmware version string changed to `0.2.1`.
- The app installs before the firmware update in the phone BLE OTA path, so the
  release APK intentionally keeps a minimal v3 telemetry bridge for update
  bootstrap.
