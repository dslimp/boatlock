# BoatLock Firmware 0.2.7 Release Notes

## Removed

- Removed the app-layer BLE pairing/auth command wrapper, owner-secret UI,
  security telemetry, and related compatibility tests.
- Removed the gated Wi-Fi HTTP `pull-update` debug deploy path. The normal
  firmware update routes are now only the release app's phone-local firmware
  file and latest GitHub Release buttons.

## Changed

- Live BLE telemetry is now binary frame v5 at 65 bytes after dropping security
  fields.
- Android/macOS app package version is now `0.2.7+7`.

## Validation

- Firmware version string changed to `0.2.7`.
- Release artifacts are still published only from `v*` tags through GitHub
  Releases.
