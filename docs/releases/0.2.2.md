# BoatLock Firmware 0.2.2 Release Notes

Release date: 2026-04-28

## Highlights

- Tagged GitHub Android APK releases now use a stable BoatLock release signing
  key from GitHub Actions secrets.
- Tagged builds fail if release signing secrets are missing, preventing
  accidental publication of an APK signed by a runner-generated debug key.
- Firmware update artifacts still use the normal `esp32s3` release profile and
  latest GitHub Release manifest path.

## Validation Snapshot

- CI helper tests: pass.
- Android release APK local build: pass.
- GitHub branch CI for `main` and `release/v0.2.x`: pass before tagging.

## Upgrade Notes

- Firmware version string changed to `0.2.2`.
- If a phone has an APK installed from the older debug-signed release, uninstall
  it once before installing this stable-signed GitHub APK. Future APK updates
  should install over the previous APK when the same release signing key is
  reused.
