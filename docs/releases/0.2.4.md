# BoatLock Firmware 0.2.4 Release Notes

## Fixed

- Android BLE OTA now defers reconnect/scan teardown while firmware bytes are
  actively uploading, so stale peer disconnects do not interrupt the GitHub
  firmware update button.
- `nh02` latest-release app-check now exercises the public GitHub Release source
  directly with `--ota-latest-release`; local temporary manifests remain
  available only when `--ota-firmware` is supplied.
- Android/macOS app package version is now `0.2.4+4`.

## Validation

- Firmware version string changed to `0.2.4`.
- Release artifacts are still published only from `v*` tags through GitHub
  Releases.
