# BoatLock Firmware 0.2.3 Release Notes

Release date: 2026-04-28

## Highlights

- Android Settings now exposes only two ESP32 firmware update paths:
  `Файл на телефоне` and `Последняя с GitHub`.
- Local firmware updates use Android's file picker, compute SHA-256 in the app,
  and then reuse the normal BLE OTA transport.
- Operator-facing firmware URL/SHA fields and app-check controls were removed
  from Settings. URL/SHA OTA is kept only for hidden bench automation.

## Validation Snapshot

- Flutter analyze: pass.
- Flutter tests: pass.
- Android release APK local build: pass.

## Upgrade Notes

- Firmware version string changed to `0.2.3`.
- On the phone, open Settings, enable `Настройка оборудования`, then use either
  `Файл на телефоне` for a local binary or `Последняя с GitHub` for the latest
  published release firmware.
